# Missions visibilité — Test Civique France

Ce dossier contient les guides et checklists pour augmenter la visibilité de Test Civique France en dehors du code.

## Objectif global
**Être visible partout** où les candidats à la naturalisation/titre de séjour recherchent de l'information.

## Roadmap d'implémentation

### Semaine 1 (Actions immédiates)
- [x] Google Search Console : vérification propriété + soumission sitemaps
- [ ] Indexation manuelle des 20 meilleurs articles
- [ ] Créer comptes réseaux sociaux (TikTok, Facebook, Instagram, YouTube)

### Semaine 2-3 (SEO systématique)
- [ ] Appliquer checklist SEO sur les 68 articles existants
- [ ] Optimiser métadonnées (title, description, keywords)
- [ ] Enrichir top 10 articles (FAQ, tableaux, infographies)
- [ ] Corriger erreurs indexation Google Search Console

### Semaine 3-4 (Réseaux sociaux)
- [ ] Lancer TikTok/Reels : 10 vidéos courtes (30-60s)
- [ ] Créer page Facebook + rejoindre groupes naturalisation
- [ ] YouTube : 3 vidéos longues (examens blancs, guides complets)
- [ ] LinkedIn : 5 posts sur titres de séjour professionnels

### Semaine 5+ (Croissance & monétisation)
- [ ] Google Publisher Center : soumission (si éligible)
- [ ] Google Discover : optimisations avancées
- [ ] Partenariats écoles de langue (DELF, TCF)
- [ ] AdSense ou alternatives monétisation
