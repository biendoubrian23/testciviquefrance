## 🗂️ Plan détaillé (sections H2)
1. **Où passer l'examen civique : les centres agréés** — agréés par France Éducation international (FEI) et la CCI Paris Île-de-France ; comment trouver le centre le plus proche.
2. **Créer son espace personnel « Examen civique »** — formulaire de pré-inscription en ligne, e-mail, confirmation par lien, mot de passe.
3. **Le document indispensable : le numéro AGDREF** — où le trouver (récépissé, titre de séjour), pourquoi il est exigé.
4. **Combien ça coûte** — prix ~70 €, variable selon le centre ; modalités de paiement indiquées par le centre.
5. **Délais et places : anticipez** — s'inscrire idéalement ≥ 2 mois avant le dépôt du dossier ; forte variabilité Paris / IDF.
6. **Le jour J** — déroulé de l'épreuve (40 questions QCM, 80 % requis), durée, ce qu'il faut apporter.
7. **Bien se préparer pour réussir du premier coup** — éviter de repayer une session → CTA produit.

## 📊 Données / faits à intégrer (vérifiés mai 2026)
- Centres agréés : **France Éducation international** + **CCI Paris Île-de-France**.
- Pré-inscription **en ligne** via un espace personnel « Examen civique ».
- **Numéro AGDREF** obligatoire à l'inscription.
- Prix : **~70 €**, non fixé uniformément par l'État (varie selon le centre).
- Anticiper **≥ 2 mois**.
